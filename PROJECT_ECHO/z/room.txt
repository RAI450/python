The player progresses from the bottom to the top, unlocking higher security levels along the way.





Each room has a distinct purpose:

Room	Main Purpose	Story Reward
Reception Hall	Learn the game	Discover what Helix was
Security Office	Restore surveillance	See your father alive in the last footage
Medical Bay	Learn about the accident	Discover he was exposed to Echo
Storage Room (Optional)	Bonus exploration	Find hidden shipping records
Research Laboratory	Major mystery reveal	Read your father's personal notes
Server Room	Discover the lockdown truth	Learn it was activated manually
Reactor Core	Restore power	Find out your father ordered the lockdown
Command Center	Final revelation	Watch your father's final message
Escape Tunnel	Escape	Leave with the truth—and one last unsettling mystery


Room 1 — Reception Hall
Purpose

This is where the player learns the controls and gets familiar with the facility.

Description

When the lockdown activates, you find yourself standing in the reception area.

Dust covers every surface.

The emergency lights flash red.

Employee desks are still arranged neatly.

Coffee cups remain untouched.

It feels as if everyone disappeared in an instant.

Objects
Reception Desk
Visitor Register
Employee Lockers
Information Board
Security Door
Puzzle

Find the login password for the reception terminal.

The password is hidden somewhere in the reception area.

Reward
Facility Map
Flashlight
Level-1 Access Card
Evidence File #1

Welcome to Helix

A company brochure introducing the Helix Research Facility.

It proudly claims:

"PROJECT ECHO will change the future forever."


Room 2 — Security Office
Description

Rows of security monitors cover one wall.

Most screens display static.

One camera flickers on for a second before turning black.

A large security terminal waits for administrator access.

Objects
Camera Console
Security Terminal
Evidence Cabinet
Weapon Locker (Locked)
Access Door
Puzzle

Restore the camera system by entering the correct security code.

Reward
Security Override Code
Master Key
Evidence File #2

Security Camera Recording

The final recording shows scientists running through the hallway.

One of them...

looks exactly like your father.

Room 3 — Medical Bay

Room 5 — Research Laboratory

This is the heart of the mystery.

Description

Father's Desk
Laboratory Terminal

Reward

Research Notes

It remembers.

Room 8 — Command Center

The final truth.

Description

This room controls the entire facility.

A giant holographic display lights up as you enter.

One AI terminal is still active.

Objects
AI Terminal
Security Database
Facility Map
Emergency Exit Controls
Puzzle

Complete a final security authentication using information gathered throughout the game.

Reward

Emergency Exit Unlocked

Evidence File #8

Final Video Message

Your father explains:

Project Echo became self-aware.
The lockdown was intentional.
He stayed behind to prevent Echo from escaping.
He hid the map and access card hoping that one day someone he trusted—you—would uncover the truth.
Final Room — Escape Tunnel
Description

The emergency tunnel leads upward toward daylight.

Warning lights flash one last time.

The massive blast door waits ahead.

One final keypad blocks your escape.

Final Puzzle

Use everything you've collected:

Access Cards
Passwords
Security Codes
Evidence Files
Research Notes

to unlock the final door.

Final Ending

You step out into the sunlight.

The mountain falls silent behind you.

Your father's sacrifice saved millions.

As you walk away, your handheld device suddenly receives one final transmission.

UNKNOWN SIGNAL DETECTED...

SOURCE:
HELIX RESEARCH FACILITY

MESSAGE:

"Hello...

Can you hear me?"

The screen fades to black.
